## Descrição

Nest é um framework <a href="http://nodejs.org" target="_blank">Node.js</a> para a construção aplicações do lado servidor de forma eficiente e escalável. Utilizando JavaScript moderno, é construído com  <a href="http://www.typescriptlang.org" target="_blank">TypeScript</a> (preserva compatibilidade com JavaScript puro) e combina elementos de POO (Programação Orientada a Objetos), PF (Programação Funcional), and PFR (Programação Funcional Reativa).

<p>Por baixo dos panos, Nest faz uso do <a href="https://expressjs.com/" target="_blank">Express</a>, mas também, fornece compatibilidade com uma ampla gama de outras bibliotecas, como por exemplo o <a href="https://github.com/fastify/fastify" target="_blank">Fastify</a>, permitindo o fácil uso com a enorme quantidade de plugins de terceiros que estão disponíveis.</p>

## Filisofia

<p>Nos últimos tempos, graças ao Node.js, o JavaScript se tornou a “lingua franca” da web, tanto para aplicações no front, quanto no backend, dando origem a projetos incríveis como <a href="https://angular.io/" target="_blank">Angular</a>, <a href="https://github.com/facebook/react" target="_blank">React</a> and <a href="https://github.com/vuejs/vue" target="_blank">Vue</a> os quais agilizam a produtividade do desenvolvedor e permitem a construção de aplicações front-end rápidas, testáveis ​​e extensíveis. Porém, no lado do servidor, embora existam muitas bibliotecas, utilitários e ferramentas excelentes para o Node, nenhum deles resolve efetivamente o problema principal - a arquitetura.</p>
<p>
O objetivo do Nest é fornecer uma arquitetura de aplicação pronta para uso, que permite a criação sem esforço de aplicações altamente testáveis, escaláveis, com baixo acoplamanto e de fácil manutenção. A arquitetura é fortemente inspirada no Angular.
</p>
